# 11.1.1.1. Felix COM API Tutorial<a href="#felix-com-api-tutorial" class="headerlink" title="Permalink to this headline">¶</a>

This tutorial will show you how to get started scripting Felix via the COM API. We’ll be using Microsoft Word’s <a href="https://web.archive.org/web/20181005073650/http://en.wikipedia.org/wiki/Visual_Basic_for_Applications" class="reference external">VBA</a> as the tutorial environment.

<div id="start-word-and-open-visual-basic" class="section">

## 11.1.1.1.1. Start Word and Open Visual Basic<a href="#start-word-and-open-visual-basic" class="headerlink" title="Permalink to this headline">¶</a>

Start MS Word, and open the Visual Basic for Applications editor. You can do this by pressing ALT+F11, or from the Developer tab, clicking **Visual Basic**.

</div>

<div id="create-a-new-module" class="section">

## 11.1.1.1.2. Create a New Module<a href="#create-a-new-module" class="headerlink" title="Permalink to this headline">¶</a>

In the left pane, there should be a “Normal” node. Expand this, and right click on the “Modules” folder.

From the context menu, select **Insert** \>\> **Module**. This is where we’ll be putting in our code.

</div>

<div id="write-macro-to-launch-felix" class="section">

## 11.1.1.1.3. Write Macro to Launch Felix<a href="#write-macro-to-launch-felix" class="headerlink" title="Permalink to this headline">¶</a>

Now, let’s write a couple of macro routines.

First, add this routine to the module you just created. It launches Felix and makes it visible:

<div class="highlight-python">

    Sub LaunchFelix()

        Dim felix As Object
        Set felix = CreateObject("Felix.App")
        felix.Visible = True

    End Sub

</div>

Now, put your cursor somewhere in the macro, and press F5 to run it. Felix should now be launched and visible, if it wasn’t already.

</div>

<div id="add-some-translations-to-the-tm" class="section">

## 11.1.1.1.4. Add Some Translations to the TM<a href="#add-some-translations-to-the-tm" class="headerlink" title="Permalink to this headline">¶</a>

Next, we’ll add a couple of translation entries.

Add the following macro to your module, and run it in the same way:

<div class="highlight-python">

    Sub AddMemoryEntries()

        Dim felix As Object
        Set felix = CreateObject("Felix.App")
        felix.Visible = True

        ' add some entries
        felix.AddMemoryEntry "source 1", "trans 1", "context 1"
        felix.AddMemoryEntry "source 2", "trans 2", "context 2"

    End Sub

</div>

Check the Felix window. There should be two entries in the TM.

</div>

<div id="add-a-glossary-entry" class="section">

## 11.1.1.1.5. Add a Glossary Entry<a href="#add-a-glossary-entry" class="headerlink" title="Permalink to this headline">¶</a>

Next, add a glossary entry. The procedure is very similar to adding TM entries:

<div class="highlight-python">

    Sub AddGlossaryEntries()

        Dim felix As Object
        Set felix = CreateObject("Felix.App")
        felix.Visible = True

        ' add some entries
        felix.AddGlossaryEntry "gloss 1", "abc", "context 1"

    End Sub

</div>

You should now have one entry in your glossary.

</div>

<div id="save-the-tm-and-glossary" class="section">

## 11.1.1.1.6. Save the TM and Glossary<a href="#save-the-tm-and-glossary" class="headerlink" title="Permalink to this headline">¶</a>

Finally, we’ll save the TM and glossary we created. Add and run this macro:

<div class="highlight-python">

    Sub SaveMemoryAndGlossary()

        Dim felix As Object
        Set felix = CreateObject("Felix.App")
        felix.Visible = True

        ' Create the gloss and mem objects
        Dim mem, gloss As Object
        Set mem = felix.App2.Memories.Item(1)
        Set gloss = felix.App2.Glossaries.Item(1)

        ' Save the TM and glossary
        mem.SaveAs "c:\my_tm.ftm"
        gloss.SaveAs "c:\my_gloss.fgloss"

    End Sub

</div>

Check your C drive; you should find the TM “my_tm.ftm”, and the glossary “my_gloss.fgloss”.

See the <a href="https://web.archive.org/web/20181005073650/http://felix-cat.com/media/manuals/felix/scripting/samples.html" class="reference internal"><em>Felix COM API Code Samples</em></a> for more sample code, and <a href="https://web.archive.org/web/20181005073650/http://felix-cat.com/media/manuals/felix/scripting/objects.html" class="reference internal"><em>Felix COM API Objects</em></a> for a list of all the available objects.

</div>
