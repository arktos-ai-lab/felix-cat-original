# 2. Getting Started<a href="#getting-started" class="headerlink" title="Permalink to this headline">¶</a>

This section describes how to download, install, and uninstall Felix.

<div id="downloading" class="section">

## 2.1. Downloading<a href="#downloading" class="headerlink" title="Permalink to this headline">¶</a>

The latest version of Felix can be downloaded from the <a href="https://web.archive.org/web/20181022230646/http://felix-cat.com/download/" class="reference external">Felix download page</a>.

The same installer is used for both the demo and full versions. Entering a license key (see <a href="#register-felix" class="reference internal"><em>Registering Felix</em></a>) in the demo version will convert it to the full version. If you upgrade Felix by installing the latest version, any license key you have already registered will be preserved.

</div>

<div id="installation" class="section">

<span id="id1"></span>

## 2.2. Installation<a href="#installation" class="headerlink" title="Permalink to this headline">¶</a>

Once you have downloaded the latest version, double click on the installer, and follow the on-screen instructions.

We recommended that you exit all other programs before proceeding, including resident anti-virus programs.

<div id="choosing-components" class="section">

### 2.2.1. Choosing Components<a href="#choosing-components" class="headerlink" title="Permalink to this headline">¶</a>

On the Select Components page of the installer, you can select to install the interfaces for MS Word, Excel, and PowerPoint (these are selected by default). This will enable you to use the Felix interface from these programs.

<div class="admonition tip">

Tip

It won’t cause an error if you select a component and that program isn’t installed. If you have multiple versions of Office installed, the component will be installed for all versions.

</div>

</div>

<div id="registering-felix" class="section">

<span id="register-felix"></span><span id="index-0"></span>

### 2.2.2. Registering Felix<a href="#registering-felix" class="headerlink" title="Permalink to this headline">¶</a>

When you first download and install Felix, it is in Demo mode. The only difference between Demo mode and Full mode is that Demo mode has a maximum TM and glossary size of 500 records. You can unlock Full mode by registering Felix.

To register Felix, you must purchase a license key. Once you have received a license, from the Felix menu, select *<span class="accelerator">H</span>elp ‣ <span class="accelerator">R</span>egister*... Next, copy and paste the key into the text box, and click **OK**.

</div>

</div>

<div id="uninstallation" class="section">

<span id="index-1"></span>

## 2.3. Uninstallation<a href="#uninstallation" class="headerlink" title="Permalink to this headline">¶</a>

The following instructions will walk you through the uninstallation process.

1.  Quit out of Microsoft Word, Excel, and PowerPoint if they are running.
2.  From the Start menu, select Control Panel.
3.  Double click on the Add/Remove Programs or Add or Remove Programs icon.
4.  Select “Felix,” then click Add/Remove or Change/Remove.
5.  The uninstall program launches. Follow the on-screen instructions to complete the uninstallation.

</div>

</div>
