The Web Interface¶

Memory Serves provides a Web interface. From it, you can:

  • Upload, view, browse, search, and delete memories and glossaries
  • View, edit, and delete translation/glossary records
  • Configure Memory Serves

Adding Memories/Glossaries¶

From the Memories or Glossaries page, click Upload Memory/Glossary. From the upload page, browse to select the desired memory or glossary file, and then click Upload.

You can also upload a memory/glossary and merge it will an existing one. Go to the View page for the memory/glossary in question, and click Upload and Merge

There’s also a stand-alone program for importing translation memories and glossaries into Memory Serves. See Memory Importer for details.

Browsing Memories/Glossaries¶

From the Memories or Glossaries page, click the Browse link for the desired memory/glossary. This link is also on the details page for each memory/glossary.

From the browse list, you can edit, delete, and view details for records.

Searching and Replacing¶

From the Memories or Glossaries page, click the Search link for the desired memory/glossary. This link is also on the details page for each memory/glossary.

See Search and Replace for more details about search and replace in Memory Serves.

Downloading Memories/Glossaries¶

You can download memories and glossaries in Felix or TMX format. Just go to the View page of the memory/glossary in question, and click Download or Download as TMX

Configuring Memory Serves Preferences¶

You can configure the preferences Memory Serves uses to calculate translation and glossary matches. See the Set Preferences section of the admin interface instructions for details.
