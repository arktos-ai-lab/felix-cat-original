Memory Serves Help

This page provides instructions on using Memory Serves with Felix.

More detailed help can be found in the online manual.

Getting Started

Since you're reading this help file, I'll assume that you managed to install Memory Serves and get it running. The next thing you need to do is get some memories/glossaries loaded into the server.

You can either upload existing Felix memories, or create new ones.

  • Upload a Felix memory
  • Create a new memory
  • Upload a Felix glossary
  • Create a new glossary

You can also configure the search preferences if you like.

Edit search preferences

Connecting from Felix

Copy the connection string of the desired memory/glossary, then from the Felix File menu, select Connect..., and paste in that connection string. The server memory/glossary should behave just like a local one, except that you don't need to save it (the server takes care of that automatically).

Searching and Browsing Memories/Glossaries

From the Memories or Glossaries page, click on the Browse or Search link for the desired memory/glossary. These links are also on the details page for each memory/glossary.

From the browse list and search results, you can edit and delete records, and view details for records.

I Forgot My Admin Password!

If you forget your admin password, you can clear the users from the database and create a new admin account using the Clear Users utility. On Windows XP, this utility will be found from the Start menu, under All Programs >> Memory Serves >> Clear Users.

The next time you start Memory Serves, it will take you to the welcome page, and you'll be able to create a new admin account. Don't forget it this time!
