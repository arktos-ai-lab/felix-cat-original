# README #

AnalyzeAssist is a tool for analyzing files to be translated (source files) against existing translation memories. It will find the amount of repetitions and fuzzy matches in the source files, to assist in estimating translation time and effort.

It supports the following types of input files:

- Text
- XML
- HTML
- MS Office (Word, PowerPoint, Excel)

AnalyzeAssist depends on the `segmenter` module:

https://bitbucket.org/ginstrom/segmenter
