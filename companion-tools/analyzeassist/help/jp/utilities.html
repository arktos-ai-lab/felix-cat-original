Command Line Utilities

A number of command-line utilities are included so that you can use AnalyzeAssist more flexibly within your own workflow.

The included command-line utilities are as follows.

  • extract_segments
    Extracts segments from a list of documents and writes them to text files
  • count_words
    Counts the words and characters in the specified input documents
  • analyze_files
    Analyzes the specified documents against the specified memories
