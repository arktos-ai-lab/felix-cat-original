Analyze Assist

AnalyzeAssist is a file-analysis program for translators. It analyzes source files against supplied translation memories, and determines the rate of matches. The current version of AnalyzeAssist is 1.5.

Features

What's it good for?

Analyze Assist analyzes source documents against your translation memories, and reports the numbers of matches (exact and fuzzy) that it finds. You can use this information for estimation and scheduling purposes.

It supports the following file types:

  • MS Word (.doc, .rtf)
  • MS Excel (.xls, .csv)
  • MS PowerPoint
  • Text
  • HTML
  • XML

It supports the following memory types:

  • Felix memory files
  • TMX version 1.4 memory files

You can save the reports generated by Analyze Assist as HTML files, or print them.
